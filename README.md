# Genetically Modified Pine Forestry Plan

This Next.js site presents a comprehensive business plan for Green Horizon Forestry Inc., a venture deploying
genetically modified pine trees to deliver accelerated timber growth, premium carbon credits, and seedling licensing
revenue.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm

### Installation

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the plan.

### Production Build

```bash
npm run build
npm start
```

## Project Structure

```
app/
  layout.jsx      # Global layout and metadata
  page.jsx        # Business plan content
  globals.css     # Global styling
next.config.js    # Next.js configuration
package.json      # Dependencies and scripts
```
